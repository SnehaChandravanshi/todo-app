## вңҰ What is FlowTask?

FlowTask is not another CRUD exercise. It is a **production-grade task management platform** with a glassmorphism dark UI, real-time analytics, drag-and-drop board, and a clean REST API вҖ” built end-to-end in a single sprint.

Three views. One keyboard shortcut. Zero friction.

##  Getting Started

### Prerequisites
- Node.js `v16+`
- npm

### Clone & Run

```bash
# Clone
git clone https://github.com/SnehaChandravanshi/todo-app.git
cd todo-app
```

```bash
# Terminal 1 вҖ” Backend (port 4000)
cd backend
npm install
npm start
```

```bash
# Terminal 2 вҖ” Frontend (port 3000)
cd frontend
npm install
npm start
```

Open **http://localhost:3000** вҖ” you're in.

>  **Shortcut:** Press `Ctrl+K` anywhere to instantly open the create task modal.

##  REST API

Base URL: `http://localhost:4000/api`

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/todos` | List todos вҖ” supports `?status=&priority=&category=&search=&sort=` |
| `GET` | `/todos/stats` | Analytics data вҖ” counts, trends, breakdowns |
| `GET` | `/todos/:id` | Single todo |
| `POST` | `/todos` | Create todo |
| `PUT` | `/todos/:id` | Update todo |
| `PATCH` | `/todos/:id/toggle` | Toggle complete |
| `PATCH` | `/todos/reorder` | Persist drag-drop order |
| `PATCH` | `/todos/:id/subtasks/:sid/toggle` | Toggle subtask |
| `DELETE` | `/todos/:id` | Delete todo |
| `DELETE` | `/todos?clearCompleted=true` | Bulk delete completed |

Full API docs вҶ’ [`docs/api.md`](docs/api.md)
